About

Welcome to the National Map Corps Status Map. The purpose of this map is to show the current status of the data gathered by clunteers throught Navigator. Each colored hexagon on the map refers to the number of points edited across the United States, Puerto Rico, and US Virgin Islands. If you wish to contribute by editing features on a specific area all you have to do is to click on any hexagon and a popup with a link leading to Navigator will appear.


This data reflects the status as of 05/28/2015
