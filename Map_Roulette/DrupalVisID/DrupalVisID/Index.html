The USGS identifier, header, and footer provided in these templates are the only versions allowed. They consist of the trademarked identifier in white on a blue background. Neither the background color, nor the identifier, can be changed, and they must be applied to every web page. Use of these templates will automatically ensure that the identifier is applied correctly.

The header and footer are designed to accomodate visual and/or data intensive applications that utilize every bit of the browser's display window. Because of this, by default, the width of the header and footer are set to 100%. You can restrict the width and modify the header and footer content only as described below.
