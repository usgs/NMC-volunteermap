Instructions

Welcome to the Points In Progress Map. If you've been having trouble finding points to edit, you’ve come to the right place

By clicking on the “Generate Random Points” Button located on the left hand side of the screen in the side menu, this interactive web map allows you to search for random unedited points or points needing peer review. If you are a new editor,you'll look for red points to edit, and if you are a peer reviewer, you can edit either red or green points.

After generating a random point, click on it to find a link that will open the TNMCorps editor where you can edit the point in question.
